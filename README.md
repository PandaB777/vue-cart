## 知识点
  * `vue`指令：`v-on` `v-if` `v-bind` `v-show` `v-for` `v-text`
  * 过滤器`filter`
  * 组件`component`
## 购物车实现
  * 创建`vue`实例
  * `v-for`渲染商品列表
  * 使用`filter`对金额和图片进行格式化

## 运行
1. 在项目终端进入bin文件

   ``` 
   cd bin 
   ```
2. 启动本地服务
    ```
    node www
    ```
3. 在浏览器输入
    ```
    http://localhost:3000/cart.html
    ```
